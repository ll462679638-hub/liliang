# ACE 网球场预约

一个无需后端即可运行的球场预约小程序原型，配置有 1 个网球场和 2 个匹克球场，包含日期、场地、时段选择，以及本地预约记录和取消预约功能；不包含支付。

## 运行

在当前目录启动任意静态文件服务，然后打开 `index.html`。例如：

```bash
python3 -m http.server 4173
```

浏览器访问 `http://localhost:4173`。

## 手机使用

项目已支持 PWA。发布到任意 HTTPS 静态网站后，苹果手机可用 Safari 的“添加到主屏幕”，安卓手机可用 Chrome 的“安装应用”，安装后会以独立应用窗口运行并支持离线打开。
